attrdict 模块，允许创建属性风格访问的字典类, python中访问嵌套字典的键值属性，语法冗长, 借助这个库，可以实现类似 JavaScript 访问嵌套对象属性的效果。

data[“Schedule”][“events”][“speakers”][0][“name”] => data.Schedule.events[0].name
